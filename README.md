## 🧭 技术栈 · N-E-X-U-S

团队名称的每一个字母，都精准锁定了一条黄金 AI 赛道：

| 字母 | 全称 | 方向 |
|---|---|---|
| **N** | Next-generation | 前沿技术信仰 —— 紧跟最新 Paper，不做过时研究 |
| **E** | Engineering | AI Infra —— 分布式训练、算力优化、显存节省 |
| **X** | X-Agents | Agent & Humanoid Robotics —— 大模型驱动的具身智能 |
| **U** | Ubiquitous Systems | 推荐算法 —— 影响亿万级用户的泛在分发系统 |
| **S** | Scale | LLM Post-Training —— SFT、RLHF / PPO 模型对齐与微调 |

## 🛠️ 本仓库 / 网站

纯静态站点，零构建依赖，通过 **GitHub Pages** 托管。

```
nexus/
├── index.html   # 单页主站
├── style.css    # 设计系统（NVIDIA 风格 · 黑白灰 + #76b900）
├── script.js    # 导航 / 滚动交互
└── README.md
```

**本地预览**：直接用浏览器打开 `index.html` 即可，或

```bash
python3 -m http.server 8000   # 然后访问 http://localhost:8000
```
